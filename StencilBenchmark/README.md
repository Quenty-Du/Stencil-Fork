#### Как запустить проект?
1. Клонируйте репозиторий
2. Скачайте набор изображений с Google Disk по ссылке: https://drive.google.com/drive/folders/1he_OPHm_fen3Hkt-zUyWBuNzXwkDBozn?usp=sharing
3. Откройте папку `StencilBenchmark` как проект в Visual Studio
4. Выберите файл sample/samples.cpp
5. Установите нужное вам значение переменной `count_of_pixels_k`. Посмотреть возможные значения можно в наборе изображений. Это числовые значения в названиях изображений.
6. Запустите собранное приложение `sample_program.exe`
7. После завершения работы программы посмотрите результат ее работы в папке resources/output


#### Примечание:
Не лишним будет отметить, что проект собирается с помощью CMake, а также использует библиотеку OpenCV. С ними могут возникнуть трудности, это решается в настройках Visual Studio.
