#### Как запустить проект?
1. Клонируйте репозиторий
2. Скачайте набор изображений с Google Disk по ссылке: https://drive.google.com/drive/folders/1he_OPHm_fen3Hkt-zUyWBuNzXwkDBozn?usp=sharing
3. Откройте папку `StencilBenchmark` как проект в Visual Studio
4. Выберите файл sample/samples.cpp
5. Установите нужное вам значение переменной `count_of_pixels_k`. Посмотреть возможные значения можно в наборе изображений. Это числовые значения в названиях изображений. (Обратите внимание, вам нужен файл .bmp, там такой пока только один, я еще не успел поменять и убрать старые)
6. Запустите скрипт (не сработает через Git Bash) `build_to_debug.bat`
7. В скрипте `increase_image_for_graphics.bash` укажите интересующую вас величину `max_pixel_count` (>= 100 000).
8. Запустите этот скрипт.
9. После завершения работы программы посмотрите результат ее работы в папке resources/output.


#### Примечание:
Проект собирается с помощью CMake.
Результаты предыдущей работы программы удаляются (логи в том числе, в будущем)
